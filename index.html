Introduction

Css is a rule based language made to put on some styling on our html by selecting the desired element we wish to make fancy.

From CSS1 to CSS3, selectors - ways and means to selecting/choosing elements on our document - have evolved as they got more and more defined as a structure or a condition through which the computer determines which element in the document corresponds the most.

And now we can be either very specific or vague and very varied in our selection, as it goes from selecting an element or a description of an element or a group of elements we would like to select
